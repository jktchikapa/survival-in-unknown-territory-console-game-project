# Survie en Terre Inconnue

Jeu développé dans le cadre du cours **420-1A6-ST – Algorithmique et programmation structurée (A25)**.

## Description
Jeu de survie où le joueur explore un monde inconnu après un accident d’avion.  
L’objectif est de collecter des ressources, fabriquer des matériaux et construire une maison à la position **[0,0]** avant l’arrivée de l’hiver.

## Fonctionnalités principales
- Déplacement sur une carte générée dynamiquement
- Terrains conservés en mémoire après exploration
- Collecte de ressources selon le type de terrain
- Inventaire consultable en tout temps
- Système de fabrication à la base
- Sauvegarde et chargement de la partie
- Menu principal (jouer, sauvegarder, charger, crédits)

## Objectifs pédagogiques
- Utilisation des listes
- Modularité du code
- Validation des entrées
- Sérialisation des données
- Implémentation d’une boucle de jeu

## Technologies
- Langage : **C#**
- Application console / jeu 2D

## Conditions de victoire
Le jeu se termine lorsque le joueur construit une maison complète à la base **[0,0]**.

## Projet scolaire
Travail pratique réalisé en équipe dans un contexte académique.
